Module: bot-express

bot-express module. This module should be mounted on webhook URI and requires configuration as options parameter.
Parameters:
Name Type Description
options Object Configuration of bot-express.
Properties
Name Type Attributes Default Description
line_channel_secret String <optional>
 LINE Channel Secret. Required when you use LINE.
line_channel_access_token String <optional>
 LINE Channel Access Token. Required when you use LINE.
facebook_app_secret String <optional>
 Facebook App Secret. Required when you use Facebook Messenger.
facebook_page_access_token Array.<Object> <optional>
 Array of a pair of Facebook Page Id and Page Access Token. Required when you use Facebook Messenger.
Properties
Name Type Description
page_id String Facebook Page Id.
page_access_token String Facebook Page Access Token.
facebook_verify_token String <optional>
 options.facebook_app_secret Facebook token to verify webook url. This is only used in initial webhook registration.
nlp_options Object NLP Configuration.
Properties
Name Type Attributes Default Description
client_access_token String Token to access to NLP service.
language String <optional>
 "ja" Language to recognize.
default_skill String <optional>
 Skill name to be used when we cannot identify the intent. Default is builtin echo-back skill which simply reply text response from NLP.
beacon_skill Object <optional>
 Skill to be used when bot receives beacon event.
Properties
Name Type Attributes Description
enter String <optional>
 Skill to be used when bot receives beacon enter event.
leave String <optional>
 Skill to be used when bot receives beacon leave event.
follow_skill String <optional>
 Skill to be used when bot receives follow event.
unfollow_skill String <optional>
 Skill to be used when bot receives unfollow event.
default_intent String <optional>
 "input.unknown" Intent name to be returned by NLP when it cannot identify the intent.
memory_retention Number <optional>
 6000 Period to retain context data in bot memory in milli-seconds.
skill_path String <optional>
 "./skill/" Path to the directory which contains skill scripts.
auto_translation String <optional>
 Flag to enable auto translation. Set this value to "enable" to enable auto translation. When set to "enable", you need to set options.google_project_id and GOOGLE_APPLICATION_CREDENTIALS environment variables.
google_project_id String <optional>
 Google Project Id to be used when you want to enable auto translation.
