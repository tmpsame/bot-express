parser(value, context, resolve, reject) → {Promise}

Function which is applied to the message from user to validate the value. You can do not only validating the value but also manipulation.
Parameters:
Name Type Description
value String | Object Data to parse. In case of text message, its text will be set in string. In case of postback event, data payload will be set in string. In other cases, message object will be set as it is.
context context Context information.
resolve function Method to call when parser judges the value fits the parameter. Return the parsed value as its argument.
reject function Method to call when parse judges the value does not fit the parameter. Optionally return the reason as its argument.
